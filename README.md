# Enterprise Cart

A production-ready enterprise eCommerce platform built using the MERN stack.

## Tech Stack

### Frontend
- React 19
- TypeScript
- Redux Toolkit
- RTK Query
- React Router
- Tailwind CSS

### Backend
- Node.js
- Express.js
- TypeScript
- MongoDB
- Mongoose

## Architecture

- Monorepo (pnpm + Turborepo)
- Feature-Based Architecture
- Clean Architecture
- Repository Pattern
- Service Layer
- SOLID Principles
